语法基础

字符串

1. 定义

在python中，字符串使用 "" 或者 '' 括起来，两者并没有什么区别。

2. 索引

字符串的索引值：

  • 从左往右 ： 0 开始
  • 从右往左 ： -1 开始

3. 切片

str[start:end:step]

  • start : 起始位置，包含
  • end ： 终点位置，不包含
  • step ：步长
4. 拼接
  • * ， 使用 str * n 可以将字符串 str 拼接 n 次
  • + ， 在两个字符串中使用，可以拼接两个字符串

import

  • 导入整个第三方库： import math（第三方库名称）
  • 为导入的库取别称： import numpy as np（别名）
  • 在库中导入相对应的方法： from math import sqrt
  • 从库中导入多个方法： from math import 方法1, 方法2, 方法3

lambda 函数

1. 函数特点
  • lambda 函数是匿名的，不需要使用 def 定义函数名称
  • 只能通过赋值给变量，或者作为参数传递给其他函数来使用
  • 通常只包含一行代码
2. 语法格式
  • lambda 是python的关键字，用于定义
  • arguments 是参数列表，可以是任意数量的参数
  • expression 是一个表达式，计算返回的值
3. 联合使用

lambda 函数通常与内置函数如 map()、filter() 和 reduce() 一起使用，以便在集合上执行操作。例如：

  • map()

  • filter()

  • reduce()
