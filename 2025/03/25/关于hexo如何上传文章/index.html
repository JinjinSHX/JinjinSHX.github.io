关于修改已经上传的文件

  1. 现在本地修改markdown文件。
  2. 运行 hexo clean 去清理已经生成的静态文件（public）和缓存文件（db.json）。
  3. 运行 hexo generate 重新生成静态文件。
  4. 运行 hexo deploy 进行重新部署。
